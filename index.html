Xcratch LINE Webhook

Xcratch の Extension Loader で読み込むための拡張です。 使う URL は dist/lineWebhook.mjs です。

読み込みURL

GitHub Pages に公開したら、次の形の URL を Xcratch に入力します。

https://YOUR-ACCOUNT.github.io/YOUR-REPOSITORY/dist/lineWebhook.mjs

Xcratch での使い方

  1. Xcratch Editor を開く
  2. 「拡張機能を追加」を押す
  3. Extension Loader を選ぶ
  4. 上の URL を入力する
  5. 拡張のブロックを使う

補足

実際の LINE 送信には授業用に管理しているサーバー側設定が必要です。 この公開ページでは、Xcratch から読み込む拡張本体のみ配布しています。
